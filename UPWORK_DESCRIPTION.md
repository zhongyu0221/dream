# Dream Garden - Portfolio Project Description

## Project Overview

**Dream Garden** is an elegant, full-stack dream journal web application that combines modern web technologies with AI-powered content generation. Users can upload images, narrate their dreams through voice input, and let AI automatically organize and save their dream entries with intelligent summaries and titles.

## Key Features

- Real-time voice-to-text transcription with bilingual support (Chinese/English)
- AI-powered dream summarization and title generation using Google Gemini
- Interactive conversational interface for enhanced dream recording
- Responsive design with elegant animations and particle effects
- Local image storage and SQLite database management

## Technical Highlights

### Frontend
- **Next.js 14** with App Router for modern React architecture
- **TypeScript** for type-safe development
- **Tailwind CSS** for responsive, utility-first styling
- **Canvas API** for dynamic particle effects and image processing
- **Web Speech API** for browser-native voice recognition

### Backend & Database
- **Next.js API Routes** for serverless backend functionality
- **Prisma ORM** for type-safe database operations
- **SQLite** for lightweight, local data persistence

### AI Integration
- **Google Gemini 2.5 Flash** for natural language processing
- Intelligent content translation and summarization
- Context-aware conversation handling

### Development Tools
- Modern build pipeline with ESLint and PostCSS
- Optimized image handling with Next.js Image component
- Custom CSS animations and Framer Motion for smooth interactions

## Technical Achievements

- Implemented real-time speech recognition with interim and final result handling
- Built AI-powered content generation pipeline with multilingual support
- Created dynamic particle effects using Canvas API for visual appeal
- Designed responsive UI with elegant typography (Cormorant Garamond) and gradient effects
- Developed RESTful API architecture for seamless frontend-backend communication

## Project Stack Summary

**Frontend:** Next.js 14, React 18, TypeScript, Tailwind CSS, Canvas API  
**Backend:** Next.js API Routes, Prisma ORM  
**Database:** SQLite  
**AI:** Google Gemini 2.5 Flash  
**APIs:** Web Speech API, Google Generative AI  
**Styling:** Tailwind CSS, Custom CSS Animations, Framer Motion
