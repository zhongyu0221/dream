# 快速启动指南

## 第一步：安装 Node.js

如果还没有安装 Node.js，请先安装：
- 访问 https://nodejs.org/
- 下载并安装 LTS 版本

## 第二步：安装依赖

在项目目录下运行：

```bash
npm install
```

## 第三步：配置环境变量

创建 `.env` 文件（在项目根目录）：

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

**注意**：如果没有 OpenAI API 密钥，应用仍然可以运行，但 AI 整理功能将不可用。你可以：
1. 访问 https://platform.openai.com/ 注册并获取 API 密钥
2. 或者暂时跳过此步骤，应用会使用原始文本作为内容

## 第四步：初始化数据库

```bash
npx prisma generate
npx prisma db push
```

## 第五步：启动开发服务器

```bash
npm run dev
```

## 第六步：访问应用

打开浏览器访问：http://localhost:3000

## 使用流程

1. **上传图片**：点击上传区域，选择一张图片作为梦境封面
2. **开始录音**：点击"开始讲述"按钮，允许浏览器使用麦克风权限
3. **讲述梦境**：用中文描述你的梦境，系统会实时显示文字
4. **停止录音**：再次点击按钮停止录音
5. **保存梦境**：点击"保存梦境"按钮，AI 会整理内容并保存
6. **查看梦境**：点击导航栏的"我的梦境"查看所有记录

## 常见问题

### 语音识别不工作？
- 确保使用 Chrome 或 Edge 浏览器
- 检查浏览器是否允许麦克风权限
- 确保网络连接正常（语音识别需要网络）

### AI 整理功能不工作？
- 检查 `.env` 文件中的 `OPENAI_API_KEY` 是否正确配置
- 确保 API 密钥有效且有足够的额度
- 查看浏览器控制台是否有错误信息

### 图片上传失败？
- 确保 `public/uploads` 目录存在且有写入权限
- 检查图片文件大小（建议小于 10MB）

## 生产环境部署

### 使用 Vercel 部署（推荐）

1. 将代码推送到 GitHub
2. 在 Vercel 导入项目
3. 添加环境变量 `OPENAI_API_KEY`
4. 部署完成

### 使用其他平台

确保：
- Node.js 版本 >= 18
- 设置环境变量
- 运行 `npm run build` 构建项目
- 运行 `npm start` 启动生产服务器
